POKEDEX

Most PokeDex apps I've come across are either laggy or cluttered with too much information. My needs were simple: I needed to know my Pokemon's Type and which Types their abilities are Super Effective against. So, I built just that. I also took the opportunity to teach myself something new, which is how to pull API information via Fetch, using only Vanilla JavaScript.

WEATHER DASHBOARD

A straightforward and easy to read app to search for up-to-date weather conditions in various locations. Previously searched for cities are stored locally, and forecasts are displayed five days in advance.

DAILY TASK MANAGER

A simple day-of planner, which displays the time of day via a live clock in the header. Timesolts that have passed by are grey, current timeslots are green, and time still left in my day is displayed as pink.

RANDOM PASSWORD GENERATOR

Regularly choosing a new password can be taxing (cyber security is important). My password generator allows me to customize how complex I need my new password to be, as well as how long it is.

RANDOM NAME GENERATOR

I play a lot of single-player video games, many of which allow me to customize several character names. To prevent time taken away from gaming, I can now click a button and a random name will be provided for me. No more wasted time trying to come up with cute names for my farm animals!
